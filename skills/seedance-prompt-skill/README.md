## 目标定位

有参考素材输入（图片、视频、音频）进行复刻、编辑、延长 或 纯文本15s内的简短视频生成（无需做分镜多段生成时）

## 核心能力

- 纯文本生成
- 一致性控制
- 运镜与动作精准复刻
- 创意模板/特效复刻
- 剧情创作/补全
- 视频延长
- 视频编辑
- 声音控制（支持音色参考、对白生成、音效设计）
- 一镜到底
- 音乐卡点（让画面节奏与音乐节拍精准匹配）

## 平台限制

- 不支持上传含有写实真人脸部的素材（图片和视频均不可），系统会自动拦截
- 文本输入有最大800汉字限制，需要优化提示词，减少字符数，同时保证效果

## 五大视频类型定义
每种类型均包含：风格特征、节奏模式、常用镜头语言、提示词结构模板、子类型

| 类型 | 子类型示例 |
|------|----------|
| 动漫视频 | 仙侠/修真、热血战斗、古风唯美、日系番剧、机甲科幻 |
| 科幻视频 | 赛博朋克、太空科幻、末日废土、近未来科技 |
| 电影特效 | 动作特效、魔法/奇幻、灾难片、微缩/微观 |
| 商业广告 | 产品展示、场景化、特效创意、美食 |
| 品牌宣传 | 企业品牌片、产品品牌片、情感品牌片、城市/地产 |
