# 视频类型参考 (Video Types Reference)

本文件定义了各视频类型的风格特征、节奏模式和提示词策略。选择视频类型后，按照对应的特征和模板进行提示词编写。

---

## 动漫视频

### 风格特征
- **画面质感**：3D国漫CG渲染 / 日漫赛璐璐 / 中国风水墨
- **色调倾向**：高饱和鲜明色彩，战斗场景偏红金/蓝紫，抒情场景偏柔光暖调
- **光影处理**：强调粒子特效、能量光效、法阵光圈；非战斗场景用丁达尔效应、体积光
- **音效特点**：配乐燃感强，战斗配BGM节奏紧凑；文戏配古风/空灵配乐

### 节奏模式
- 战斗类：短促快切 + 关键动作升格慢放，节奏"快-快-慢-快"
- 文戏类：中景对话缓推 + 环境全景过渡，节奏舒缓
- 变身/觉醒类：蓄力（慢）→ 爆发（快）→ 定格展示（慢）

### 常用镜头语言
- 环绕摇镜快切（战斗展示）
- 仰拍 + 升格（角色觉醒/变身）
- 大远景 → 特写推进（出场）
- 一镜到底（连续追逐/战斗）

### 提示词结构模板
```
[画面风格总纲] + [角色描述@图片N] + [动作序列（含时间戳）] + [特效描述] + [音效/台词] + [镜头语言]
```

### 子类型
| 子类型 | 关键差异 | 推荐风格 |
|--------|----------|----------|
| 仙侠/修真 | 法术特效、仙山云海 | 3D国漫CG、中国风水墨 |
| 热血战斗 | 高速动作、能量爆发 | 超精细CG、红金高饱和 |
| 古风唯美 | 服饰飘逸、意境留白 | 水墨风、莫兰迪色系 |
| 日系番剧 | 表情夸张、速度线 | 日漫赛璐璐 |
| 机甲/科幻动漫 | 机械变形、光束武器 | CG渲染+赛博朋克 |

## 科幻视频

### 风格特征
- **画面质感**：高精度CG渲染、未来科技UI、全息投影元素
- **色调倾向**：冰蓝冷调、霓虹赛博、深蓝夜幕；偶尔用黑金质感
- **光影处理**：霓虹反射、全息光、激光束、能量场发光；环境多暗调+局部高亮
- **音效特点**：电子合成音、低频嗡鸣、机械运转声、全息界面操作音

### 节奏模式
- 探索类：缓慢推进、长镜头揭示未知空间
- 追逐/战斗类：快切+速度感的运动模糊
- 悬疑/揭示类：慢节奏铺垫 → 突然加速的反转

### 常用镜头语言
- 第一人称/主观视角（穿越空间）
- 航拍/鸟瞰（展示未来城市）
- 希区柯克变焦（营造不安）
- 跟拍 + 甩镜转场

### 提示词结构模板
```
[技术规格（画幅/帧率）] + [环境世界观] + [主体及装备描述] + [事件/动作] + [科技特效描述] + [音效设计] + [镜头语言]
```

### 子类型
| 子类型 | 关键差异 | 推荐风格 |
|--------|----------|----------|
| 赛博朋克 | 霓虹都市、人机混合 | 赛博朋克、霓虹高饱和冷暖对比 |
| 太空科幻 | 星际飞船、宇宙空间 | 深蓝夜幕、好莱坞大片 |
| 末日废土 | 荒芜废墟、生存感 | 焦黄暖调+青灰冷调交替 |
| 近未来科技 | 日常科技增强、AI交互 | 极简主义+冰蓝冷调 |

## 电影特效

### 风格特征
- **画面质感**：电影级宽银幕（2.35:1）、24fps、浅景深、胶片质感
- **色调倾向**：根据题材灵活变化，但始终追求电影级调色
- **光影处理**：高度讲究布光层次，主光+辅光+轮廓光的专业组合
- **音效特点**：沉浸式音效设计，环境声+动效+配乐层次分明

### 节奏模式
- 动作类：慢蓄力 → 快爆发 → 升格定格 → 恢复常速，循环往复
- 悬疑类：长镜头缓推积累紧张感 → 突然的视角切换
- 史诗类：大远景铺陈 → 逐步推近至特写 → 最终全景收束

### 常用镜头语言
- 斯坦尼康稳定跟拍
- 升降镜头（crane shot）营造仪式感
- 遮挡擦镜转场
- 慢动作 + 正常速度交替

### 提示词结构模板
```
[技术规格（2.35:1/24fps/时长）] + [调色风格] + [场景环境] + [角色及动作（时间戳分镜）] + [光影设计] + [音效层次] + [镜头语言] + [禁止项]
```

### 子类型
| 子类型 | 关键差异 | 推荐风格 |
|--------|----------|----------|
| 动作特效 | 爆炸、追逐、格斗 | 好莱坞大片、高动态范围 |
| 魔法/奇幻特效 | 法术、变身、元素控制 | CG渲染+体积光 |
| 灾难片特效 | 自然灾害、大场面破坏 | 写实CG+手持摇晃 |
| 微缩/微观特效 | 微观世界、分子级画面 | 超精细CG、4K医学CGI |

## 商业广告

### 风格特征
- **画面质感**：高清棚拍感、产品精准呈现、画面干净利落
- **色调倾向**：根据品牌调性选择，通常高饱和+高明度；高端品常用黑金/极简
- **光影处理**：棚拍灯光精准打光，侧逆光勾勒产品轮廓；强调材质质感
- **音效特点**：节奏明快的BGM、产品交互音效（开盖声/倒水声/点击声）

### 节奏模式
- 产品展示类：稳定匀速旋转/推拉 → 爆炸分解 → 合成定格
- 场景代入类：生活场景切入 → 产品特写 → 结尾品牌露出
- 卡点广告类：跟随音乐节拍快切多角度展示

### 常用镜头语言
- 360度环绕旋转（产品全方位展示）
- 微距特写（材质/细节）
- 升降+推进（品牌LOGO收尾）
- 爆炸分解/三维拆解

### 提示词结构模板
```
[产品主体@图片N] + [展示动作序列] + [材质/细节描述] + [背景环境] + [品牌元素] + [音效/BGM节奏] + [镜头语言]
```

### 子类型
| 子类型 | 关键差异 | 推荐风格 |
|--------|----------|----------|
| 产品展示广告 | 聚焦产品本身 | 商业摄影+极简背景 |
| 场景化广告 | 产品融入生活场景 | 电影摄影+暖调 |
| 特效创意广告 | 3D变形、爆炸分解 | CGI+超现实主义 |
| 美食广告 | 食材特写、烹饪过程 | 暖调+微距+慢动作 |

## 品牌宣传

### 风格特征
- **画面质感**：兼顾情感叙事与品牌调性，画面精致有质感
- **色调倾向**：与品牌VI色系呼应；通常偏高级感（莫兰迪/黑金/低饱和暖调）
- **光影处理**：自然光为主导，结合环境叙事；讲究黄金时段/柔光/大气光效
- **音效特点**：配乐情感驱动，旁白温暖有力，环境声增加真实感

### 节奏模式
- 叙事类：起承转合的故事节奏，从困境到解决方案到品牌展示
- 愿景类：大远景→中景→特写，逐步揭示品牌理念
- 情感类：生活片段蒙太奇，配合音乐节奏变化

### 常用镜头语言
- 稳定器跟拍（人物故事）
- 航拍+大远景（展示格局）
- 延时摄影（时间流逝/城市发展）
- 叠化转场（场景间的柔和过渡）

### 提示词结构模板
```
[品牌调性总纲] + [场景叙事（时间戳分镜）] + [人物/产品描述] + [情感氛围] + [旁白/台词] + [音乐/音效设计] + [镜头语言]
```

### 子类型
| 子类型 | 关键差异 | 推荐风格 |
|--------|----------|----------|
| 企业品牌片 | 企业实力/愿景展示 | 电影摄影+航拍+黑金 |
| 产品品牌片 | 产品故事/理念传达 | 暖调叙事+极简 |
| 情感品牌片 | 以情动人的品牌故事 | 纪录片风+自然光 |
| 城市/地产宣传 | 空间展示/生活方式 | 航拍+延时+莫兰迪 |
