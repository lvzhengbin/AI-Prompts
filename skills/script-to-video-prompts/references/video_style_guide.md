## 主流视频风格

### 写实/电影风格

| 风格 | 英文关键词 | 特征 | 适用 |
|------|-----------|------|------|
| **电影级** | cinematic, filmic, movie-like | 宽画幅、浅景深、调色 | 剧情片 |
| **纪录片** | documentary style, realistic | 自然光、手持、真实感 | 纪实内容 |
| **高清写实** | photorealistic, hyperrealistic | 极度真实、高细节 | 商业广告 |
| **胶片质感** | film grain, 35mm film look | 颗粒感、柔和色彩 | 复古风格 |

### 风格化/艺术风格

| 风格 | 英文关键词 | 特征 | 适用 |
|------|-----------|------|------|
| **动漫风** | anime style, Japanese animation | 大眼、鲜艳色彩、干净线条 | 动漫MV |
| **3D卡通** | 3D cartoon, Pixar style | 圆润、夸张、可爱 | 儿童内容 |
| **水彩画** | watercolor style, painted | 柔和边缘、流动色彩 | 艺术片 |
| **油画风** | oil painting style, impressionist | 笔触感、厚重色彩 | 艺术内容 |
| **复古** | vintage, retro, nostalgic | 褪色、颗粒、旧色调 | 怀旧内容 |
| **赛博朋克** | cyberpunk, neon noir | 霓虹、暗色、高科技 | 科幻题材 |
| **蒸汽波** | vaporwave, synthwave | 粉紫色、80s美学 | 音乐视频 |

### 技术风格

| 风格 | 英文关键词 | 特征 | 适用 |
|------|-----------|------|------|
| **慢动作** | slow motion, high speed | 时间延展、细节可见 | 动作场景 |
| **延时** | timelapse | 时间压缩 | 时间流逝 |
| **微距** | macro, close-up detail | 极近距离、细节 | 产品展示 |
| **航拍** | aerial, drone footage | 高空视角、大场面 | 风景场景 |

## 风格关键词详解

### 光线相关

```
自然光系列:
- natural lighting: 自然光
- golden hour: 黄金时刻光
- soft diffused light: 柔和漫射光
- harsh sunlight: 强烈阳光
- overcast lighting: 阴天光

人造光系列:
- studio lighting: 棚拍光
- neon lighting: 霓虹灯光
- candlelight: 烛光
- dramatic lighting: 戏剧性光
- rim lighting: 轮廓光
- backlit: 逆光
```

### 镜头相关

```
景深系列:
- shallow depth of field: 浅景深（背景虚化）
- deep focus: 深焦（全清晰）
- bokeh: 焦外光斑
- soft focus: 柔焦

镜头效果:
- lens flare: 镜头光晕
- anamorphic: 变形宽银幕
- wide angle: 广角
- telephoto: 长焦压缩
```

### 色彩相关

```
色调系列:
- warm tones: 暖色调
- cool tones: 冷色调
- muted colors: 柔和色彩
- vibrant colors: 鲜艳色彩
- desaturated: 低饱和
- high contrast: 高对比

调色风格:
- teal and orange: 青橙调色（好莱坞常用）
- bleach bypass: 银留效果
- cross-processed: 交叉冲洗
```

### 画面质感

```
清晰度系列:
- sharp: 锐利
- crisp: 清脆
- detailed: 细节丰富
- 4K, 8K: 分辨率标注

质感系列:
- film grain: 胶片颗粒
- noise-free: 无噪点
- clean: 干净
- textured: 有质感
```

## 视频特定参数

### 画面比例

| 比例 | 英文 | 用途 |
|------|------|------|
| 16:9 | widescreen, 16:9 aspect ratio | 标准视频 |
| 2.35:1 | cinemascope, anamorphic widescreen | 电影感 |
| 1:1 | square format | 社交媒体 |
| 9:16 | vertical video, portrait | 短视频 |
| 4:3 | classic TV ratio | 复古感 |

### 帧率表达

```
- 24fps: cinematic frame rate (电影感)
- 30fps: standard video (标准)
- 60fps: smooth motion (流畅)
- 120fps: slow motion capable (慢动作)
```

### 时长相关

```
- short clip: 短片段
- loop: 循环视频
- seamless loop: 无缝循环
- transition: 过渡片段
```

## 常见问题避免

### AI视频常见问题及对策

| 问题 | 原因 | 对策关键词 |
|------|------|-----------|
| 角色变形 | 一致性差 | consistent character, same person |
| 动作不自然 | 运动理解差 | natural movement, smooth motion |
| 背景抖动 | 场景不稳 | stable background, consistent environment |
| 光线跳变 | 光照不一致 | consistent lighting, stable illumination |
| 细节丢失 | 压缩或生成差 | detailed, high quality, sharp |
| 颜色闪烁 | 帧间不一致 | consistent colors, temporal coherence |

### 提示词优化建议

**避免**：
- 过长的提示词（超过200词）
- 相互矛盾的描述
- 过于抽象的概念
- 版权相关的具体作品名

**推荐**：
- 简洁明确的描述
- 具体的视觉元素
- 一致的风格关键词
- 明确的主体和动作
