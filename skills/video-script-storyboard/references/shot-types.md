# 镜头类型参考 (Shot Types Reference)

## 景别分类 (Shot Sizes)

| 类型 | 英文 | 缩写 | 描述 | 适用场景 |
|------|------|------|------|----------|
| 大远景 | Extreme Wide Shot | EWS | 展示整体环境，人物渺小 | 开场建立氛围、史诗感 |
| 远景 | Wide Shot | WS | 展示完整场景与人物 | 场景交代、空间关系 |
| 全景 | Full Shot | FS | 展示人物全身 | 动作展示、角色出场 |
| 中景 | Medium Shot | MS | 腰部以上 | 对话、叙事主体 |
| 中近景 | Medium Close-Up | MCU | 胸部以上 | 访谈、产品展示 |
| 近景 | Close-Up | CU | 面部或局部特写 | 表情、情感传递 |
| 特写 | Extreme Close-Up | ECU | 极度局部（眼睛、嘴唇） | 细节强调、情绪高潮 |
| 过肩镜头 | Over-the-Shoulder | OTS | 从一方肩膀越过拍另一方 | 对话场景 |

## 运镜方式 (Camera Movements)

| 运镜 | 英文 | 描述 | 情绪效果 |
|------|------|------|----------|
| 推 | Push In / Dolly In | 镜头前推接近主体 | 聚焦、紧张感增强 |
| 拉 | Pull Back / Dolly Out | 镜头后退远离主体 | 揭示、孤独感 |
| 摇 | Pan (L/R) | 水平旋转 | 跟随、扫视环境 |
| 仰/俯 | Tilt (Up/Down) | 垂直旋转 | 仰视=威严，俯视=渺小 |
| 跟 | Tracking / Follow | 摄影机跟随移动 | 跟随角色行动 |
| 航拍 | Aerial / Drone | 空中俯瞰 | 宏观、壮阔 |
| 手持 | Handheld | 轻微抖动感 | 真实、紧张、紧迫 |
| 轨道 | Orbit / Arc | 围绕主体环绕 | 英雄出场、全方位展示 |
| 升降 | Crane / Boom | 垂直升降 | 场景转换、仪式感 |
| 变焦 | Zoom In/Out | 镜头变焦 | 突然聚焦、戏剧效果 |
| 旋转 | Rotate / Dutch Angle | 倾斜/旋转 | 不安、迷幻、超现实 |

## 转场方式 (Transitions)

| 转场 | 英文 | 适用场景 |
|------|------|----------|
| 硬切 | Hard Cut | 最通用，快节奏 |
| 叠化 | Dissolve | 时间流逝、抒情过渡 |
| 淡入/淡出 | Fade In/Out | 开头/结尾、段落转换 |
| 闪白/闪黑 | Flash White/Black | 回忆、冲击感 |
| 甩镜 | Whip Pan | 快速转换、动感 |
| 遮挡 | Wipe / Object Pass | 遮挡物过镜后切换场景 |
| 匹配剪辑 | Match Cut | 通过视觉相似元素衔接 |
| J-Cut / L-Cut | J-Cut / L-Cut | 声音先行/延续的剪辑 |
