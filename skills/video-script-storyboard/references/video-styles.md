# 视频风格参考 (Video Styles Reference)

## 视觉风格 (Visual Styles)

| 风格 | 关键词 | 适合类型 |
|------|--------|----------|
| 电影感 (Cinematic) | 宽银幕、浅景深、调色、光影层次 | 品牌宣传、叙事短片 |
| 动漫风 (Anime) | 赛璐珞、鲜艳色彩、夸张表情、速度线 | 动漫二次元、IP内容 |
| 写实纪录 (Documentary) | 自然光、手持镜头、采访剪辑 | 纪录片、Vlog |
| 赛博朋克 (Cyberpunk) | 霓虹灯、雨夜、暗色调、未来城市 | 科技、游戏 |
| 复古 (Retro/Vintage) | 胶片颗粒、饱和度偏低、暖色调 | 怀旧、文艺 |
| 极简 (Minimalist) | 大量留白、简洁构图、干净字体 | 产品、科技 |
| 3D渲染 (3D Render) | 光滑质感、环境光、反射材质 | 产品展示、概念设计 |
| 水墨/国风 (Chinese Ink) | 水墨晕染、留白、山水 | 国风、传统文化 |
| Comic (漫画) | 漫画分格、对话气泡、网点阴影 | 趣味科普、故事 |
| 定格动画 (Stop Motion) | 实物逐帧、手工质感 | 创意、教育 |
| 扁平设计 (Flat Design) | 几何形状、无阴影、纯色块 | MG动画、解说 |
| 涂鸦/手绘 (Doodle) | 手绘线条、白板动画风格 | 教育、趣味解说 |

## 色调/视觉氛围 (Color Tones & Moods)

| 氛围 | 色调描述 | 情绪效果 |
|------|----------|----------|
| 温暖治愈 | 橙黄暖色、柔光、低对比 | 亲切、安全 |
| 冷调高级 | 蓝灰冷色、高对比 | 专业、疏离 |
| 高饱和活力 | 鲜艳多彩、高饱和 | 年轻、动感 |
| 暗黑深沉 | 低明度、阴影为主 | 悬疑、恐怖 |
| 柔和梦幻 | 粉紫柔光、朦胧 | 浪漫、幻想 |
| 高对比戏剧 | 明暗对比极端 | 紧张、戏剧性 |
| 自然清新 | 绿色调、自然光 | 清新、治愈 |
| 霓虹闪烁 | 荧光色、光晕 | 赛博、夜生活 |

## 常见视频类型节奏对应

| 视频类型 | 建议节奏 | 镜头时长 | 特点 |
|----------|----------|----------|------|
| 短视频/Reels | 极快 | 1-3s | 高信息密度、钩子前置 |
| 品牌宣传片 | 中等偏慢 | 3-6s | 质感优先、情绪渲染 |
| 教程/解说 | 中等 | 3-8s | 节奏服务于信息传达 |
| 叙事短片 | 慢 | 4-10s | 叙事节奏、情绪递进 |
| 音乐MV | 随BPM | 1-4s | 节拍对齐、高频切换 |
| 产品展示 | 中等 | 3-5s | 突出产品细节 |
