# 场景类型参考 (Scene Types Reference)

## 场景分类

### 室内场景 (Interior)

| 场景 | 关键词 | 适合内容 |
|------|--------|----------|
| 办公室 | 工位、白板、会议室、落地窗 | 商务、科技 |
| 家居 | 客厅、卧室、厨房、书房 | 生活方式、Vlog |
| 工作室 | 灯光设备、录音棚、暗房 | 创意、音乐 |
| 实验室 | 仪器、屏幕、白大褂 | 科技、教育 |
| 咖啡馆 | 暖光、木质桌椅、咖啡杯 | 文艺、社交 |
| 商场 | 橱窗、灯光闪烁、购物 | 消费、时尚 |
| 展厅/博物馆 | 聚光灯、展品、空旷 | 文化、品牌 |

### 室外场景 (Exterior)

| 场景 | 关键词 | 适合内容 |
|------|--------|----------|
| 城市街头 | 霓虹、行人、车流 | 都市、街拍 |
| 自然风光 | 山川、湖泊、森林、日落 | 旅行、治愈 |
| 海边/沙滩 | 浪花、沙滩、蓝天 | 休闲、运动 |
| 工业区 | 工厂、烟囱、钢铁结构 | 纪录、工业风 |
| 田园/农村 | 稻田、老屋、炊烟 | 乡村、美食 |
| 校园 | 操场、教室、图书馆 | 教育、青春 |

### 虚拟/概念场景 (Virtual/Conceptual)

| 场景 | 关键词 | 适合内容 |
|------|--------|----------|
| 虚拟空间 | 数据流、粒子、全息 | 科技、AI |
| 宇宙/太空 | 星系、星云、空间站 | 科幻、探索 |
| 幻想世界 | 魔法、城堡、异兽 | 奇幻、游戏 |
| 抽象空间 | 几何体、光线、无限延伸 | MG动画、概念 |
| 微观世界 | 细胞、分子、晶体结构 | 科普、教育 |
| 像素世界 | 8bit、像素化、复古游戏 | 游戏、怀旧 |

## 场景描述模板

描述场景时，按以下要素组织：

```
[环境] + [光线] + [氛围] + [关键道具/元素]
```

**示例：**
- "雨夜的赛博朋克城市街头，霓虹灯倒映在湿漉漉的路面上，蒸汽从下水道口升起"
- "阳光透过落地窗洒入极简风格的白色工作室，桌上摆放着一台开着的笔记本电脑"
- "深空中漂浮的空间站，地球在背景中缓慢旋转，冷蓝色环境光"
