# Seedance 2.0 分镜提示词模板库

本文件提供各类场景的完整分镜提示词示例，按视频类型分组。生成时：
1. 先选择匹配的示例作为参考框架
2. 替换角色、场景、时长等具体参数
3. 根据是否长视频决定使用「单段模板」或「多段衔接模板」

---

## 📐 提示词结构说明

### 单段视频（≤15秒）标准结构

```
【整体描述】风格 + 时长 + 画面比例 + 整体氛围
首帧@首帧参考图，角色@角色参考图（如有）

0–Xs：[运镜方式]，[画面内容]，[主体动作]，[对白/旁白]，[光影/特效]
X–Ys：[运镜方式]，[画面内容]，[主体动作]，[对白/旁白]，[光影/特效]
...

【声音】配乐风格 / 音效 / 对白
【参考素材】（如有）@图片1 用途说明，@视频1 用途说明
```

### 多段长视频衔接结构（>15秒）

```
【多宫格分镜概览生图提示词】
生图提示词：一张包含 3 个宫格的分镜故事板图片，排版为 1行3列。
@角色名 @场景名，保持美术风格一致。
宫格1：[分镜1尾帧关键画面描述]
宫格2：[分镜2尾帧关键画面描述]
宫格3：[分镜3尾帧关键画面描述]

整体描述 + 角色引用 + 风格 + 氛围

0–5s：...
```
