# 🎬 Seedance Storyboard for long video

将任何剧本（视频时长大于15秒）生成多个分镜提示词，支持全能参考和首尾帧两种模式生成视频。确保在生成的多个视频中保持人物，场景的一致性和连续的叙事。

## 功能特点

### 🎯 核心能力
- ✅ **分步引导** - 从想法到完整提示词的完整流程
- ✅ **视频分镜** - 将长视频拆解为多个分镜提示词
- ✅ **高级技巧** - 复杂运镜复刻、角色一致性控制

### 🚀 补充要求
- 🔥 **复杂运镜** - 希区柯克变焦、环绕跟拍、一镜到底
- 🔥 **特效复刻** - 变装特效、粒子特效、拼图转场
- 🔥 **角色一致性** - 保持人物形象的连续叙事
- 🔥 **多场景融合** - 无缝转场、空间转换


### ⚠️ 平台限制

- **不支持上传含有写实真人脸部的素材**（图片和视频均不可），系统会自动拦截
- 有参考视频时生成消耗更多
- 视频延长时，选择的生成时长应为「新增部分」的时长
- 文本输入有最大800汉字限制，需精炼提示词保证效果

## 分步引导流程

### 第一步：理解用户想法

用友好的方式询问用户：
1. **你想讲什么故事？**（一句话概括核心内容）
2. **视频时长？**（默认15秒，如果是15s就生成单视频，否则就生成多段视频）
3. **横屏/竖屏？**（默认横屏16:9）
4. **生成模式？**（全能参考和首尾帧两种模式生成视频）
5. **参考素材情况？**（有无可用的图片/视频）

### 第二步：深入挖掘细节

根据用户的初步回答，逐步引导完善以下维度：

#### 1. 内容叙事
- 故事的起承转合是什么？
- 关键情节有哪些？
- 需要哪些角色/主体？
- 对白/旁白内容？

#### 2. 视觉风格
- 整体风格：写实/动画/水墨/科幻/复古/电影感？
- 色调氛围：明亮/昏暗/暖色/冷色/黑白？
- 画面比例：竖屏(9:16)/横屏(16:9)/电影宽屏(2.35:1)？

#### 3. 镜头语言
- 景别变化：远景→全景→中景→近景→特写？
- 运镜方式：推/拉/摇/移/跟/环绕/升降/希区柯克变焦？
- 转场方式：硬切/渐变/匹配剪辑/特效转场？

#### 4. 动作与节奏
- 主体动作：行走/奔跑/打斗/舞蹈/表情变化？
- 动作节奏：舒缓/急促/有张有弛？
- 是否需要音乐卡点？

#### 5. 声音设计
- 配乐风格：激昂/温馨/悬疑/欢快/史诗？
- 音效需求：环境音/动作音效/特殊音效？
- 对白/旁白：有台词吗？什么语气？

### 第三步：构建分镜结构 和 生成专业提示词

#### 3.1 角色特征 和 主场景环境提示词
For each main character, generate a standalone AI image prompt:
> [Character Name]: [Detailed visual description, clothing, art style, lighting, quality tags]

main scene/environment, generate a standalone AI image prompt:
> [Scene Name]: [Environment, atmosphere, lighting, color palette, art style, quality tags]

#### 3.2 分镜片段提示词

将故事拆解为时间轴上的镜头序列：

时长15秒内视频示例结构：
```
0-3秒：开场镜头，建立场景
3-6秒：发展，引入主体/冲突
6-10秒：高潮，核心动作/情感爆发
10-13秒：转折/过渡
13-15秒：结尾/落版
```

时长超过15秒的视频要构建一个多宫格的分镜提示词 和 多个单独的分镜提示词，每个分镜提示词的时长最长为15秒。每个分镜提示词之间要保持人物，场景的一致性和连续的叙事。
【**重点要求**：每个分镜图片都必须为该片段的尾帧，这样下一个分片分镜视频的生成就可以利用@上一个分片的尾帧作为首帧 和 @当前分片的尾帧作为尾帧，生成更连贯的视频】
- 先生成一个多宫格的分镜提示词（提示词中要引用前面已确定了的@角色和@场景的图片资源，确保生成的多宫格内容人物和场景与前面确定的一致）
- 再生成多个单独的分镜提示词（每个分镜提示词的时长最长为15秒，每个分镜提示词之间要保持人物，场景的一致性和连续的叙事）

#### 提示词结构模版

**提示词结构模板**：
参考@templates/storyboard-template.md 文档内容
```
【分镜1描述】
【分镜1内容整体描述】首帧@首帧图片，尾帧@尾帧图片，角色@角色图片，风格 + 时长 + 画面比例 + 整体氛围
0-X秒：[镜头运动]，[画面内容]，[主体动作]，[对白/旁白]，[光影/特效]
X-Y秒：[镜头运动]，[画面内容]，[主体动作]，[对白/旁白]，[光影/特效]
...

【分镜2描述】
【分镜2内容整体描述】首帧@首帧图片，尾帧@尾帧图片，角色@角色图片，风格 + 时长 + 画面比例 + 整体氛围
0-X秒：[镜头运动]，[画面内容]，[主体动作]，[对白/旁白]，[光影/特效]
X-Y秒：[镜头运动]，[画面内容]，[主体动作]，[对白/旁白]，[光影/特效]
...

【声音说明】配乐风格/音效/对白

【参考素材说明】（如有）
@图片1 作为首帧/角色参考
@视频1 参考运镜/动作
@音频1 用于配乐/对白参考
```

**提示词书写原则**：
1. 时间轴清晰，标注每段的秒数范围
2. 镜头语言明确（推/拉/摇/移/跟/环绕等）
3. 动作描述具体，避免模糊词汇
4. 多模态引用时使用 @素材名 格式
5. 需要延长视频时说明"延长Xs"并选择对应生成长度
